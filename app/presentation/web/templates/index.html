ML Translate

Онлайн-перевод с английского на французский и обратно с учётом баланса кредитов.

Возможности

  • Регистрация и авторизация (JWT)
  • Личный кабинет с балансом в кредитах
  • Пополнение баланса (без эквайринга)
  • Запросы к ML-модели из Web и через REST
  • Списание кредитов только при успешном предсказании
  • История переводов и транзакций
  • Очередь задач (RabbitMQ) и воркеры
  • Валидация входных данных и проверка положительного баланса
  • Роль администратора: пополнение пользователям, просмотр всех транзакций

Перейти в кабинет API Docs

Быстрый старт (REST)

Примеры запросов:

POST /auth/login
{
  "email": "user@example.com",
  "password": "userpass"
}

GET /wallet/           (Bearer токен)
POST /wallet/topup     {"amount": 100}
POST /translate        {"input_text":"hello","source_lang":"en","target_lang":"fr"}

Для Telegram-бота укажи API_BASE, API_EMAIL, API_PASSWORD в .env.
